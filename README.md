# Analysis of the Spectrum of Random Matrices

## Project Overview
This repository contains my Bachelor's Thesis in Mathematics (University of Florence), titled **"Analysis of the spectrum of random matrices: from Wigner's Theorem to large deviations"**.

## Abstract
The thesis explores Random Matrix Theory, a field bridging probability, statistics, and quantum physics.
The work focuses on the convergence of the spectral measure of random matrices. Specifically, it compares two mathematical approaches:
1.  **Combinatorial Methods & Graph Theory:** Used to demonstrate Wigner's Theorem and the convergence to the standard semicircular law.
2.  **Large Deviation Theory:** Used to provide a more general result under stricter assumptions.

## Key Topics
* Random Matrices (Wigner Matrices)
* Spectral Analysis
* Semicircular Law
* Large Deviation Theory
